# Regressão Logistica binária - Depressão

Objetivo: Depressão, análise dos sintomas e fatores que favorecem sintomas depressivos

Projeto pessoal para entender um pouco o grupo de pessoas que foram diagnosticas com depressão, nesse estudo foi utilizada a base de dados do IBGE-PNS2019 (IBGE-Plano Nacional de Saúde).

Utilizado o pacote do CRAN.R PNSIBGE que disponibiliza diversas funcionalidades para o banco de dados IBGE-PNS.
O mesmo banco de dados pode ser encontrado em outros locais, como kaggle, site do IBGE e em diferentes formatos.

O dicionário de dados utilizado para estudo e entendimento das variaveis foi do site do IBGE-PNS: https://www.ibge.gov.br/estatisticas/sociais/saude/9160-pesquisa-nacional-de-saude.html?=&t=microdados
